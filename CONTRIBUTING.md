# Contributing

Thank you for helping TheRouter.ai.

## Where to contribute
- `therouter-examples`: examples and workflows
- `awesome-chinese-ai-models`: add models/providers with official links

## Submission checklist
- Include an official source link (docs/blog/post)
- Add a one-line summary of the change
- Note status (available, open-source, access level)

Docs: https://therouter.ai/docs/
